   ```
   TWILIO_AUTH_TOKEN=(seu token)
   TWILIO_WHATSAPP_NUMBER=(seu número)
   STRIPE_SECRET_KEY=(sua chave)
   PAGSEGURO_SANDBOX_EMAIL=(seu email)
   PAGSEGURO_SANDBOX_TOKEN=(seu token)
   ```
